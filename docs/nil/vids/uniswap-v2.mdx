# Uniswap v2 =nil;

This video demonstrates the key features of Uniswap v2 =nil;, a conversion of the well-known Uniswap v2 protocol to the zkSharding architecture. The project demonstrates how the zkSharding architecture can be easily applied to an existing dAppa and it also shows how zkSharding achieves parallelization.
