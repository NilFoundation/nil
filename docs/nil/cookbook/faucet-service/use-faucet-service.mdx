# Use the faucet service via Nil.js

[**Faucets**](../../getting-started/essentials/tokens-multi-currency.mdx#currency-faucet-service) form a special service for 'topping up' addresses and existing smart contracts. Existing faucets can distribute the 'default' token used to pay for message execution as well as several pre-defined 'mock' currencies such as BTC or ETH. 

:::info

'Mock' currencies from faucets are identical to custom currencies and cannot be used to pay for execution.

:::
