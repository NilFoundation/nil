# @nilfoundation/niljs

## Classes

| Class | Description |
| ------ | ------ |
| [BlockNotFoundError](classes/BlockNotFoundError.mdx) | The error class for 'block not found' errors. This error is thrown when the requested block is not found. |
| [ExternalMessageEnvelope](classes/ExternalMessageEnvelope.mdx) | The envelope for an external message (a message sent by a user, a dApp, etc.) |
| [Faucet](classes/Faucet.mdx) | Faucet is a special contract that is used to top up other contracts in the =nil; devnet. |
| [HttpTransport](classes/HttpTransport.mdx) | HttpTransport represents the HTTP transport for connecting to the network. |
| [ITransport](classes/ITransport.mdx) | The transport interface. |
| [InternalMessageEnvelope](classes/InternalMessageEnvelope.mdx) | The envelope for an internal message (a message sent by a smart contract to another smart contract). |
| [InvalidShardIdError](classes/InvalidShardIdError.mdx) | The error class for invalid shard ID. This error is thrown when the provided shard ID is invalid. |
| [LocalECDSAKeySigner](classes/LocalECDSAKeySigner.mdx) | LocalKeySigner is a class that allows for signing data with the private key. It uses the secp256k1 curve implementation by the @noble/curves/secp256k1 library. |
| [MetaMaskSigner](classes/MetaMaskSigner.mdx) | MetaMaskSigner is a class that allows for signing data using MetaMask. This signer can only be used inside a browser. |
| [MetaMaskTransport](classes/MetaMaskTransport.mdx) | MetaMask transport represents the MetaMask transport for connecting to the network. MetaMask transport can be used in browser only. MetaMaskTransport |
| [PublicClient](classes/PublicClient.mdx) | PublicClient is a class that allows for interacting with the network via the JSON-RPC API. It provides an abstraction of the connection to =nil;. PublicClient enables using API requests that do not require signing data (or otherwise using one's private key). |
| [WalletV1](classes/WalletV1.mdx) | WalletV1 is a class used for performing operations on the cluster that require authentication. |
